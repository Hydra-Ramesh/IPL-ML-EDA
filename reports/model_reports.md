# 🤖 Match Outcome Prediction Model

## 📌 Problem Statement
The objective is to predict the winner of an IPL match using historical match data.

---

## 📊 Features Used

- Batting Team
- Bowling Team
- Venue
- Toss Winner
- Toss Decision
- Team Win Percentage
- Toss Impact Feature

---

## ⚙️ Model Used

- Random Forest Classifier

Reason:
- Handles non-linear relationships
- Works well with mixed feature types
- Robust and interpretable

## 📈 Model Performance

- Accuracy: XX%
